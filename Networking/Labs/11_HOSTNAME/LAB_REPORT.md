# Lab Report

## Experiment Title

**Using the HOSTNAME Command to Display the Local Computer Name**

---

## Objective

The objective of this experiment was to understand the Windows **HOSTNAME** command and learn how to identify the local computer's hostname for network administration, troubleshooting, and cybersecurity purposes.

---

## Tools Used

- Operating System: Windows 11
- Command Prompt (CMD)
- HOSTNAME Utility

---

## Commands Executed

```cmd
hostname
hostname /?
```

---

## Procedure

1. Opened Command Prompt.
2. Executed the `hostname` command to display the current computer name.
3. Executed the `hostname /?` command to view the available help information.
4. Recorded the outputs and captured screenshots.
5. Analyzed the role of hostnames in Windows networking.

---

## Results

- Successfully displayed the system hostname.
- Verified the hostname as **Jakhmola_Akshat**.
- Viewed the built-in help information for the HOSTNAME utility.
- Learned that the command is used only to display the current host name.
- Understood how hostnames are used for device identification and network communication.

---

## Conclusion

The experiment successfully demonstrated the use of the Windows **HOSTNAME** command. It provided the hostname of the local computer and highlighted its importance in identifying devices on a network. The command is simple but useful for system administration, troubleshooting, Active Directory environments, DNS management, and cybersecurity investigations.

---

## Skills Gained

- Used the Windows HOSTNAME command.
- Identified the local computer name.
- Understood the role of hostnames in networking.
- Learned how hostnames assist in DNS and Active Directory environments.
- Improved basic Windows command-line skills.
- Developed knowledge useful for system administration and cybersecurity.
