# Lab Report

## Experiment Title

Study of the TASKLIST Command in Windows

---

## Objective

To learn how to use the TASKLIST command to enumerate running processes, display detailed process information, inspect services and DLL modules, and understand its importance in Windows administration and cybersecurity.

---

## Tools Used

- Windows 11
- Command Prompt

---

## Commands Executed

```cmd
tasklist
tasklist /v
tasklist /svc
tasklist /m
tasklist /?
```

---

## Procedure

1. Opened Command Prompt.
2. Executed the `tasklist` command.
3. Displayed verbose process information using `/v`.
4. Displayed services associated with running processes using `/svc`.
5. Displayed DLL modules loaded by processes using `/m`.
6. Viewed the built-in help using `/?`.
7. Recorded observations and captured screenshots.

---

## Results

Successfully enumerated all running processes, viewed detailed process information, identified services associated with processes, inspected loaded DLL modules, and studied the built-in documentation of the TASKLIST command.

---

## Conclusion

The TASKLIST command is an essential Windows utility for monitoring running processes and collecting process-related information. It plays an important role in system administration, troubleshooting, malware detection, incident response, and digital forensic investigations.
